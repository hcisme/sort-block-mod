# 📦 智能归档单元 (Smart Sorting Unit) 使用手册

## 简介
**智能归档单元**是一种高科技的物品收集与分类设备。它不仅能自动吸取掉落在它上方的物品，还能通过扫描周围的容器，智能地将物品分发到对应的箱子中。

它最大的特点是支持**“视觉化分类”**：你只需要用**物品展示框**或**告示牌**标记箱子，机器就会自动理解你的分类意图。

---

## 🛠️ 快速开始

1.  **放置机器**：将智能归档单元放置在地面上。
2.  **布置存储区**：在机器周围放置箱子、桶或任何模组的容器（支持范围见下文）。
3.  **标记箱子**：使用物品展示框或告示牌贴在箱子上，定义它们的用途（详见“分类规则”）。
4.  **投入物品**：将物品扔在机器**正上方**，或者通过漏斗/管道将物品输入到机器上方。机器会产生粒子光束，将物品传输到目标箱子。

---

## 📋 分类规则与优先级

机器会按照 **优先级 (Priority)** 从高到低尝试存入物品。

### 🥇 优先级 1：专用过滤器 (Filter Chest)
*   **如何设置**：在箱子上贴一个**物品展示框**，并在框内放入**指定的物品**（例如：苹果）。
*   **行为逻辑**：机器会检查掉落物是否与展示框内的物品一致。
    *   ✅ **是**：物品存入该箱子。
    *   ❌ **否**：跳过，寻找下一个箱子。
*   **用途**：用于存放大量特定资源（如矿物、农作物）。

### 🥈 优先级 2：自动学习单元 (Auto-Learning Chest)
*   **如何设置**：在箱子上贴一个**空的物品展示框**。
*   **行为逻辑**：
    1.  机器检测到展示框是空的。
    2.  机器会将当前处理的物品（例如：原木）存入箱子。
    3.  ✨ **自动贴标**：机器会自动从箱子里取出一个原木，贴在展示框上。
    4.  **结果**：该箱子从此升级为 **优先级 1 (专用过滤器)**，以后专门接收原木。
*   **用途**：无需手动设置过滤器，让系统自动根据流入的物品建立分类。

### 🥉 优先级 3：杂物/溢出箱 (Sundries/Dump Chest)
*   **如何设置**：在箱子上贴一个**告示牌**，并在告示牌的任意一行写上 `Sundries` (不区分大小写)。
*   **行为逻辑**：无视物品类型，只要箱子没满，什么都收。
*   **用途**：
    *   存放没有被归类的杂乱物品。
    *   当专用箱子（优先级 1）满了时，作为溢出缓冲区。

## ⚙️ 常见场景演示

**场景 A：箱子满了怎么办？**
1.  如果“苹果箱子”（优先级 1）满了，机器会寻找下一个苹果箱子。
2.  如果没有其他苹果箱子，或者所有苹果箱子都满了，机器会尝试放入“杂物箱”（优先级 3）。
3.  如果连杂物箱也满了，物品会保留在机器上方（或掉落在地上），等待清理。

**场景 B：自动学习功能**
你放置了一排贴着空展示框的箱子。你往机器上扔了一组钻石、一组铁锭、一组金锭。
*   机器把钻石放入第一个箱子，并把展示框变成钻石。
*   机器把铁锭放入第二个箱子，并把展示框变成铁锭。
*   ...以此类推。你不需要手动去贴每一个框！

## 📏 技术参数

*   **有效工作范围**：
    *   以机器为中心。
    *   **X轴 (东西向)**: 40 格 (左右各 20)
    *   **Y轴 (高度)**: 5 格 (上下各 2)
    *   **Z轴 (南北向)**: 8 格 (前后各 4)
    *   *可自行修改*
*   **扫描频率**：每 5 秒（100 ticks）重新扫描一次周围环境。
    *   *注意：如果你刚放下一个新箱子，机器可能需要几秒钟才能“看见”它。*
*   **兼容性**：
    *   支持原版箱子、桶、潜影盒、熔炉（燃料/原料自动判断）。
    *   支持所有基于 Fabric API 标准的模组容器（如 Tech Reborn, Modern Industrialization 等）。
